# 가사동영상생성기
midnight lemon씨를 참고해 형식의 가사영상을 생성하는 응용프로그램입니다

## 사용방법
# 1.  `input`폴더로 이동해 `settings.yml`을 열고 아래 내용을 설정합니다


`배경색1`: 배경색입니다

`배경색2`: 제목화명 중앙으로 있는 배경색입니다

`글자체 주소`: ttf, otf 등

`동영상 url`: Youtube에 있는 동영상 url



# 2. `title.yml`을 설정합니다


`조선말 제목`: 

`조선말 제목 글자 크기`: 

`일본말 제목`: 

`일본말 제목 글자 크기`: 

`작사`: 입력 없이 가능

`작곡`: 입력 없이 가능

`연주`:  입력 없이 가능

`표시시간`: 동영상 내 화상 표시를 종료하는 시간을 입력하지 않으면 동영상에 포함되지 않습니다



# 3.가사를 yml 형식으로 입력합니다


`표시시간`: 동영상 내 화상 표시를 종료하는 시간을 입력하지 않으면 동영상에 포함되지 않습니다

`가사` : 최대 4줄까지 

우에서부터 순서대로 파일이 읽어오므로 `1.yaml`,`2.yaml`와 같이 파일을 생성해 주십시오


# 4.`main.py`를 기동합니다


`output` 파일 내에 출력된 파일이 있습니다
